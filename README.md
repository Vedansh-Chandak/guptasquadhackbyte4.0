# 🛠️ FeedFix — AI-Powered Civic Hazard Detection

> Detect, report, and resolve road hazards in your community — powered by AI.

---

## 📌 Overview

**FeedFix** is an AI-powered civic hazard detection platform that enables citizens to report road hazards like potholes, cracks, waterlogging, and damaged infrastructure using their smartphones.

The system leverages AI to analyze uploaded images, automatically detect hazard types, and generate actionable reports. It aims to bridge the gap between citizens and civic authorities through real-time, data-driven reporting.

Built for **HackByte 4.0**, FeedFix focuses on making civic engagement faster, smarter, and more effective.

---

## ✨ Features

* 📸 **AI Hazard Detection**
  Upload or capture images; AI automatically detects road hazards

* 🧠 **Smart Classification**
  Categorizes hazard type and assigns severity

* 🗺️ **Geo-tagged Reports**
  Automatically attaches location data for accurate mapping

* 📊 **Live Hazard Map**
  Visualize all reported hazards on an interactive map

* 👍 **Community Feed & Upvoting**
  Users can upvote hazards to increase their visibility

* 🔔 **Status Tracking**
  Track reports from *open → in progress → resolved*

* 📱 **Mobile-Friendly UI**
  Optimized for seamless usage across devices

---

## 🧪 How It Works

1. **Capture** — User uploads or takes a photo of a hazard
2. **Analyze** — AI processes the image and detects type + severity
3. **Auto-fill** — Report is enriched with AI insights and geo-location
4. **Submit** — Data is stored and displayed on the public map
5. **Track** — Users and authorities monitor resolution status
